# Vantage Wealth — Documentation

This repo contains the documentation site for [Vantage Wealth](https://github.com/rmurarishetti/vantage), powered by [Mintlify](https://mintlify.com).

## Local preview

```bash
npm i -g mintlify
mintlify dev
```

Open [http://localhost:3000](http://localhost:3000) to preview.

## Structure

```
introduction.mdx       — What is Vantage?
quickstart.mdx         — 15-minute setup guide
setup/                 — Supabase, backend, Flutter, Android
features/              — Expense tracking, AI advisor, portfolio, etc.
ai/                    — Architecture, RAG pipeline, adding tools
self-hosting/          — EC2, Docker
legal/                 — License, CLA
```
