The AI advisor uses deterministic tools to fetch your real data. No numbers are made up â€” every answer is sourced from one of these tools.

## Cash Flow Tools

| Tool | Description |
|------|-------------|
| `get_spending_by_category` | Total spend per category for a date range. Optional category filter. |
| `get_spending_by_merchant` | Top merchants ranked by spend, with transaction count. |
| `get_spending_trend` | Monthly spend trajectory over 6+ months. |
| `get_transactions` | Individual transactions with filters (category, merchant, amount range, date). |
| `get_budget_status` | Budget vs actual comparison with % used alerts. |
| `get_recurring_transactions` | Detects subscriptions from 90-day patterns. Returns merchant, estimated monthly cost. |
| `compare_spending_periods` | Side-by-side category comparison between two date ranges. |
| `get_income_summary` | Income totals grouped by source/merchant. |

## Investment Tools

| Tool | Description |
|------|-------------|
| `get_holdings` | All portfolio positions with cost basis. Filter by asset type. |
| `get_portfolio_summary` | Total value, cost basis, total return %, allocation by type. |
| `get_asset_detail` | Single holding: quantity, cost, current price, P&L, daily change. |
| `get_market_price` | Live prices for multiple symbols (stocks, crypto, ETFs). |
| `get_historical_prices` | OHLCV data for any period (1d to max). |
| `get_allocation_breakdown` | Portfolio breakdown by asset type or currency with percentages. |
| `get_portfolio_performance` | Per-holding and total return analysis. |
| `get_concentration_risk` | Top holdings %, HHI diversification index, risk assessment. |
