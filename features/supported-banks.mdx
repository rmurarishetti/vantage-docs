## Adding a new bank

Want your bank supported? The parser is designed to be extended. Each bank needs:

1. Its Android package name (found via `adb logcat` or the Play Store URL)
2. A set of `RegExp` patterns matching its notification formats
3. A PR to `vantage_wealth/lib/core/utils/notification_parser.dart` and `VantageNotificationListener.kt`

See [CONTRIBUTING.md](https://github.com/rmurarishetti/vantage/blob/main/CONTRIBUTING.md#adding-a-new-banking-app) for the full guide with code examples.
